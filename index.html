Welcome to Comete

Comete is a hobby project developed by Alexander Nilsen. Comete enables the user to use their Android devices to control FlightGear or MSFS over wireless network with the use of built-in accelerometer sensors, or using as a instrument panel with often used gauges. Comete supersedes Comete for FlightGear and Alni FlightGear Control.

  • This project requires HoloEverywhere

About Regular Comete

The regular Comete enables the user to control FlightGear or MSFS/P3D over the local network with the accelerometer of the Android device.

The use can stear the craft, use the throttle, gears and brakes. The user can even set the COMM, NAV and transponder frequencies of the craft.

Regular Comete also includes basic information about the heading and velocity of the craft.

About Comete Gauges

Comete Gauges does not provide the user to control the simulator as the regular Comete does. But does hovever provide the user a way to connect their Android tablets to FlightGear or MSFS/P3D and use it as a instrument panel with gauges for attitude, airspeed, vertical speed, heading and altitude.

The graphics for the instruments are taken from the FlightGear project and adapted for use with Comete Gauges.
