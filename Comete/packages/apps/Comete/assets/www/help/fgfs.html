Usage for Windows hosts

or others using the FlightGear Wizard launcher (fgrun)

  1. Start the FlightGear Wizard ( fgrun )
  2. Go through the basic setup
  3. Then at the last screen before launching FlightGear, press the button at the bottom right labelled Advanced
  4. Within this window go to the category on the left labelled Network
  5. Here, enable the option called props, and then specify the port where FlightGear should listen to commands
  6. When this is done click OK

FlightGear is now ready to be used with Comete

Usage for hosts using the JFlightWizard launcher

  1. Start the Java application JFlightWizard
  2. Go through the basic setup
  3. Then select the last category on the left labelled Network Options
  4. In the option (telnet) Enabled telnet server on the specific port, specify the port that is going to be used
  5. Rememver to click the button Save before launching FlightGear

FlightGear is now ready to be used with Comete

Usage for hosts not using any launcher

  • If the FlightGear executable ( fgfs ) is in your system's PATH, open a command line and type
    • fgfs --props=<port>
    • e.g: fgfs --props=9000
  • If this is not the case, change the working directory to where the FlightGear executable resides
    • If using a *NIX based system (like Mac OS, BSD, Linux, Solaris etc.) type
    • ./fgfs --props=<port>
    • e.g: ./fgfs --props=9000
    • If using a Windows based system just type
    • fgfs --props=<port>
    • e.g: fgfs --props=9000

FlightGear is now ready to be used with Comete
