Setup MSFS

  1. Install the newest version of FSUIPC
    • You may need to register (buy) FSUIPC
  2. Get the latest archive of the Comete Server modules for FSUIPC
    • Please visit https://github.com/alni/comete/wiki/MSFS-Setup to download the archive
  3. Extract the archive to a temporary location
  4. Copy the files in the Modules folder to the Modules folder of your Flight Simulator
    • The ipcready.lua file is optional. Only copy this if you want Comete to auto-start after FSUIPC is fully loaded
    • If the ipcready.lua file already exists and you want Comete to auto-start, then open the ipcready.lua file within your FlightSim Modules in a text editor, and on a new line type/copy ipc.macro("Lua Comete-Server")
  5. Open the Comete-Server.lua file in a text editor and change the host to the IP address of your computer running the Flight Simulator and optionally the port to a free port
    • e.g. HOST = "192.168.1.10"
    • e.g. PORT = "8384"
  6. If you have chosen to auto-start Comete after FSUIPC is loaded, you need to specify at hotkey for starting Comete (this reguires a registered FSUIPC)
    • After choosing the hotkey to use, look for Lua Comete-Server within the Controls list and select it
    • You can now use that hotkey to start and restart Comete
    • Optionally choose a hotkey to be used to stop Comete (look for LuaKill Comete-Server in the Controls list)

MSFS is now ready to be used with Comete
